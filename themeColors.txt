### N++ PALETTE COLORS ###
##########################

### ------ INTRODUCTION ------

Originally, each theme has a palette folder, each palette has one tga image
per object, and each image has a horizontal sequence of 64x64 tiles, one for
each differently colored portion of the object. Actually, only the (32,31)
pixel for each block is used to check the color, the rest doesn't matter.
The colors have an alpha layer, but transparency is ignored anyway.

This file contains a list of tables. The first one is an index explaining
what each file in the palette means and how many colors it contains.
Then, each file is described in detail, with explanations for each
individual color in it.

Note: The switches for the locked and trap doors are included in the same
palette as their corresponding doors. Toggle mines are also with mines,
and micro drones use the same palette as zap drones.

# Palette locations:
  * Linux: home/<user>/.steam/steam/steamapps/common/N++/NPP
  * Windows: SteamLibrary/steamapps/common/N++/NPP
  * mac: home/<user>/Library/ ... /steamapps
